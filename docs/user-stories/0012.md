# 0012: View RSVPs and the inquiry inbox in an admin dashboard

## Story

As an administrator, I want one dashboard listing RSVPs by meeting and all inbound inquiries, so that I can track engagement and respond from a single place.

## Acceptance Criteria

- The dashboard lists RSVPs grouped by meeting, each showing name, email, and timestamp
- The dashboard lists inquiry/join/start-a-digest submissions with submitter name, email, type, message, and timestamp
- Non-admin (unauthenticated) users cannot reach the dashboard or its data
- Each list shows an empty state when no records exist
- Lists reflect newly submitted records after the page is loaded/refreshed
